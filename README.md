TellTrail is a SwiftUI iOS app for capturing and sharing location-based voice memories. The app lets users record “voice drops,” view nearby and recent trails in a feed, follow their personal journey, chat around shared experiences, and manage their profile through a clean tab-based interface.

It is organized around feature modules for Feed, Journey, Chat, Profile, and Record Voice Drop, with shared domain models, preview data, reusable UI components, and a central app coordinator for navigation and permissions.
